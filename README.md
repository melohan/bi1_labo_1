# Hello Wolrd Microservice

## Prerequisites

- Python 3.10.x


## Setting up dev

```
git clone git@github.com:melohan/bi1_labo_1.git
cd bi1_labo_1/
```

## Run

1. Activate virtual environment 
```
python3 -m venv .venv
source .venv/bin/activate
```

2. Run docker
``` 
docker run python-docker
```
